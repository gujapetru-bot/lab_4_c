Теmа: Alocarea dinamică a memoriei pentru tablourile bidimensionale. Utilizarea funcţiilor şi a pointerilor

Scopul lucrării: Programarea algoritmilor de prelucrare a tablourilor bidimensionale prin utilizarea funcţiilor, pointerilor şi alocarea dinamică a memoriei pentru tablou.

Sarcină: Pentru tabloul bidimensional dat din n linii şi m coloane să se afişeze la ecran următorul meniu de opţiuni:

Alocarea dinamică a memoriei pentru tablourile bidimensionale
Introducerea elementelor tabloului de la tastatură
Completarea tabloului cu valori aleatorii
Să se sorteze a doua şi penultima linie ale tabloului 
în ordine descendentă cu ajutorul metodei de sortare prin selecţie 
şi schimb.
Afișarea elementelor tabloului la ecran
Eliberarea memoriei alocate pentru tablou
Ieșire din program
Programul este un meniu interactiv care lucrează cu o matrice de numere întregi alocată dinamic.

Compilarea programului se realizează cu ajutorul compilatorului GCC, folosind comanda: gcc LP4.c -o program

Sau instalarea fisierului LP4.exe si run file
